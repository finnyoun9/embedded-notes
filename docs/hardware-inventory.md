## 四、逻辑分析仪使用指南

**硬件**：24MHz 8CH（Saleae 兼容，CY7C68013A），macOS 通过 `sigrok-cli` + PulseView 使用。

### 快速上手

```bash
# 1. 检测设备
sigrok-cli --scan
# 应显示：fx2lafw - Saleae Logic with 8 channels

# 2. 采集信号（例：CH1/D0 接 ESP32 GPIO2）
sigrok-cli -d fx2lafw --samples 3M -c samplerate=1M --channels D0 -o capture.sr

# 3. 生成波形图
sigrok-cli -i capture.sr -O vcd -o capture.vcd
python3 simulations/view_wave.py capture.vcd capture.png

# 4. 协议解码（在 PulseView 中 File→Open capture.sr → 加解码器）
```

### 常用协议接线

| 协议 | 通道 | sigrok-cli 参数 |
|------|------|----------------|
| I2C | CH1=SCL, CH2=SDA | `-P i2c:scl=1:sda=2` |
| SPI | CH1=CLK, CH2=MISO, CH3=MOSI, CH4=CS | `-P spi:clk=1:miso=2:mosi=3:cs=4` |
| UART | CH1=RX | `-P uart:rx=1:baudrate=115200` |

⚠️ **GND 必须接被测电路 GND，否则波形全乱。**

## 九、理论 → 实验 → 简历 映射

### 9.1 数字电路

```
课本            实验                           简历写法
─────────────────────────────────────────────────────────
组合逻辑  ──→  74HC00/138 面包板 + LA 抓真值表  「数字逻辑电路验证与调试」
时序逻辑  ──→  74HC74 分频器 + 74HC161 计数器   「时序电路设计与时序分析」
状态机   ──→   STM32 FSM + LA 验证状态跳转      「FSM 控制器硬件实现」
总线协议  ──→  I2C/SPI 多设备 + LA 协议解码     「I2C/SPI 总线协议栈与调试」
```

> ⭐ **面试核弹：「I2C 总线通信协议栈设计与时序分析」**  
> STM32 挂 2-3 个 I2C 设备 → LA 抓 SCL/SDA → PulseView 解码 → 能讲起始/停止/ACK/地址扫描/仲裁。直接关联计组「总线」章节。

### 9.2 模拟电路

```
课本            实验                           简历写法
─────────────────────────────────────────────────────────
BJT 放大  ──→  2N2222 共射放大 + 万用表测 Q 点   「分立元件放大电路设计与测试」
运放     ──→  LM358 反相/同相/滤波/比较器        「模拟信号调理电路设计」
MOSFET   ──→  IRFZ44N PWM 驱动 DC 电机            「功率开关与电机驱动电路」
二极管   ──→  DP100 扫 V-I 特性 + Python 绘图     「半导体器件特性测试平台」
```

> ⭐ **面试核弹：「基于分立元件的线性稳压电源设计」**  
> `DP100(输入) → TIP122(调整管) → LM358(误差放大) → 输出`，5.1V 稳压管做基准。教科书级串联稳压电源，面试画图讲 5 分钟。

### 9.3 计算机组成原理

| 计组章节 | 硬件实验 | 简历写法 |
|----------|----------|----------|
| 总线 | 74HC245 总线收发 + I2C/SPI 多设备 + LA 抓总线时序 | 「嵌入式总线架构设计与验证」 |
| 存储层次 | STM32 Flash vs SRAM 读写 + I2C EEPROM 外部存储 | 「嵌入式存储系统实验」 |
| 中断 | NVIC 优先级实验 + LA 测中断延迟 | 「实时系统中异常处理机制研究」 |
| I/O 接口 | 74HC573 锁存器 + 74HC138 地址译码 → 自制 I/O 扩展 | 「存储映射 I/O 接口设计」 |
