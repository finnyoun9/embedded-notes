## 一、从原理图到 PCB 的标准流程

```
画原理图  →  检查DRC  →  转PCB  →  布局  →  布线  →  铺铜  →  导出Gerber  →  下单打样
(SCH)        (无误)     (一键)    (摆放)   (连线)   (GND)    (制造文件)   (嘉立创)
```

## 二、三个入门 PCB 项目

### 项目 1：LED 闪灯板（最简单）⭐

**电路**：
```
VCC(5V) ──[R 220Ω]──[LED]── GND
                      │
                   GPIO(接MCU)
```

**学什么**：
- 双面板布线
- 丝印标注
- 5V 电源布线
- 导出 Gerber

**步骤**：
1. 新建工程 → 新建原理图
2. 从库拖：`LED`、`Resistor(220)`、`Header 2x1`(接电源)、`Header 1x3`(接MCU)
3. 连好线 → 工具 → 设计规则检查(DRC)
4. 设计 → 转换到 PCB
5. 摆放元件(别重叠) → 布线(自动或手动) → 铺铜(GND层)
6. 文件 → 导出 → Gerber → 下单

---

### 项目 2：RC 低通滤波器（有实际用途）⭐⭐

**电路**：
```
Vin ──[R 1kΩ]──┬── Vout
               [C 100nF]
                │
               GND
```

**PCB 上多两排排针**（输入/输出各一个 2pin 排针），用示波器/逻辑分析仪测。

**学什么**：
- 信号布线（Vin/Vout 走线尽量短）
- 去耦电容靠近元件
- 测试点（Test Point）添加

**元件清单**：
| 元件 | 封装 | 数量 |
|------|------|------|
| 1kΩ 电阻 | 0805 或 直插 AXIAL-0.4 | 1 |
| 100nF 电容 | 0805 | 1 |
| 2pin 排针 | 2.54mm | 2 |
| 测试点 | TP-1.5mm | 2 |

---

### 项目 3：5V→3.3V LDO 稳压板（面试项目）⭐⭐⭐

**电路**：
```
         [AMS1117-3.3]
Vin(5V) ──┤IN    OUT├── Vout(3.3V)
           │   GND  │
          [C1 10µF]  [C2 10µF]
           │         │
          GND       GND
```

这就是你仿真里画过的 LDO 稳压器！直接做成实物。

**学什么**：
- 电源 IC  layout（输入/输出电容尽量靠近 IC）
- 大面积铺铜散热
- 双层板：顶层信号+底层完整 GND
- BOM 表生成

**元件清单**：
| 元件 | 封装 | 数量 |
|------|------|------|
| AMS1117-3.3 | SOT-223 | 1 |
| 10µF 电容 | 0805 或 1206 | 2 |
| 100nF 电容 | 0805 | 2 |
| 2pin 排针 | 2.54mm | 2 |

## 三、关键操作速查

| 操作 | 怎么做 |
|------|--------|
| 搜索元件 | 原理图左侧库面板 → 搜索（如 `AMS1117`） |
| 旋转元件 | 选中 → 空格键 |
| 自动布线 | PCB 界面 → 布线 → 自动布线 |
| 铺铜 | PCB 界面 → 放置 → 铺铜区域 → 画矩形 → 选 GND 网络 |
| DRC 检查 | 工具 → 设计规则检查 |
| 导出 Gerber | 文件 → 导出 → Gerber（PCB制造文件） |
| 下单打样 | 导出后自动弹窗 → 选板厚/颜色/数量 → 付款 |

## 四、新手常见坑

| 坑 | 解决 |
|----|------|
| **忘记加 GND 符号** | 原理图里每个 GND 必须是 `power:GND` 符号，不是文字 |
| **封装选错** | 直插(THT)vs贴片(SMD)分清，AMS1117 用 SOT-223 |
| **丝印压焊盘** | 导出前检查丝印层，别让文字压在焊盘上（无法焊接） |
| **走线太细** | 电源线 ≥0.5mm，信号线 ≥0.25mm |
| **没铺铜** | 双层板底层整面铺 GND 铜，减少噪声 |
| **DRC 报错不修** | 红色叉号 = 必须修，否则打样出来是废板 |

## 五、从学习到打样：推荐路线

```
第1天：LED闪灯板 → 走通全流程，下单（¥5 包邮 5片）
第3天：收到板子 → 焊接 LED+电阻 → 用 DP100 供电 → 亮了！
第5天：RC滤波器 → 打样 → 用逻辑分析仪测频响
第7天：LDO 稳压板 → 打样 → 用万用表测 3.3V 输出 → 贴简历
```

> **¥15 打三次样，简历多三个项目。**
