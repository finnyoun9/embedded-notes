# PCB 设计学习路径 — 从零画出 STM32 最小系统板

> **目标**: 画一块 STM32F103C8T6 最小系统板，下单打样（嘉立创 5 元包邮），面试展示"我会 PCB 设计"
> **工具**: KiCad 8（已安装在 Mac）+ 立创EDA（Web 版作为辅助）

---

## 为什么学 PCB？

深圳嵌入式岗位中，**"能看懂原理图 + 会画 PCB"是硬件方向的硬技能**。即使你做软件岗，画过 PCB 也是简历亮点——证明你不是只会写代码的纯软工程师。

## 学习路线（2 周）

### Week 1: 理论 + 工具上手

```
Day 1-2: KiCad 界面熟悉
  ├── 新建工程 → 打开原理图编辑器
  ├── 放置元件: 电阻(R)、电容(C)、LED、排针

  └── 电气规则检查 (ERC)

  ├── 电源: AMS1117-3.3V (5V→3.3V)
  ├── 时钟: 8MHz 晶振 + 32.768KHz RTC 晶振
  ├── 复位: 按键 + RC 电路
  ├── 调试: SWD 接口 (SWCLK/SWDIO + 3.3V/GND)
  ├── BOOT0/BOOT1 启动选择

Day 5: 原理图 Review + ERC 通过
```

### Week 2: PCB Layout + 打样

```
Day 6-7: PCB Layout
  ├── 板框设置 (5cm × 5cm, 双面板)
```

## STM32F103C8T6 最小系统关键电路

### 电源部分
```
USB 5V ──→ AMS1117-3.3V ──→ VDD (3.3V)
              │                  │
          10µF+100nF          100nF×4 (每个 VDD 引脚)
              │
             GND
```

### 时钟
```
OSC_IN  ── 8MHz 晶振 ── OSC_OUT
  │            │
22pF         22pF
  │            │
 GND          GND
```

### SWD 调试接口
```
1: VDD (3.3V)  ←→ ST-Link Pin 1
2: SWCLK        ←→ ST-Link Pin 2
3: GND          ←→ ST-Link Pin 3
4: SWDIO        ←→ ST-Link Pin 4
```

## 关键知识点（面试会问）

| 问题 | 答案要点 |
|------|----------|
| **去耦电容为什么每个 VDD 都要放？** | 滤除数字电路高频噪声，靠近引脚放置降低环路电感 |
| **晶振负载电容怎么选？** | 查晶振 datasheet 的 CL 值，通常 12-22pF；CL = (C1×C2)/(C1+C2) + Cstray |
| **覆铜的作用？** | 大面积 GND 覆铜 → 降低地阻抗、散热、EMI 屏蔽 |
| **为什么用 3.3V？** | STM32 供电范围 2.0-3.6V，3.3V 是标准逻辑电平 |
| **BOOT0/BOOT1 怎么接？** | BOOT0 拉低(Flash 启动) + 留跳线帽切换到高(串口下载) |

## 立创EDA 快速出板流程（替代 KiCad）

1. 打开 https://lceda.cn/editor
2. 新建工程 → 原理图
3. 搜索 "STM32F103C8T6" → 放入封装
4. 按上面最小系统图连线
5. 一键转 PCB → 自动布线 → 下单

**对比**: 立创EDA 快（1-2 天出板）但 KiCad 学习价值高（面试加分更多）。
